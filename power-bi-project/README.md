# Artificial Jewelry Sales Analysis (Power BI)

## Project Overview
This project focuses on analyzing artificial jewelry sales data to understand customer behavior, sales trends, and overall business performance using Power BI.

## Tools Used
- Microsoft Excel
- Power BI
- Tableau

## Key Insights
- The West region recorded the highest sales volume
- Category B contributed the most to total revenue
- Sales showed peaks during January, May, and August
- The average discount was approximately 14.42%
- A high return rate indicates possible issues in product quality or delivery

## Key Metrics
- Total Sales: 288,106
- Average Discount: 14.42%
- Profit: 248,525

## Features
- Interactive dashboard for data visualization
- Region-wise and category-wise analysis
- Monthly sales trend analysis
- KPI-based insights for decision making
